# Set MSU base R plotting palette

Set MSU base R plotting palette

## Usage

``` r
set_msu_palette(palette)
```

## Arguments

- palette:

  Name of palette. See `names(MSUthemes::MSUpalettes)`.

## Value

Returns a character vector giving the colors from the palette which was
in effect. This is invisible unless the argument is omitted.

## Examples

``` r
set_msu_palette("msu_qual1")
```
